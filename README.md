# jour_377_data_story
Data Story project for JOUR 377 - Intro to Data Journalism

# Subdirectories

`clean_data`: Full cleaned and joined dataset, generated with `scripts`.

`memos`: Contains the pitch memo.

`plots`: Contains all plots generated with `scripts`.

`raw_data`: Raw data scraped from Fangraphs.com.

`scripts`: R scripts used to acquire and process data, and produce visualizations.
